# wsz-vscode README

自用vscode 前端插件, 包含自用主题，fileheader(沿用koro1FileHeader), console (turbo)插件

默认功能键

### 输出console 变量
  "key": "ctrl+cmd+[",
  "mac": "ctrl+cmd+["

### 注释console 变量
  "key": "ctrl+alt+/",
  "mac": "ctrl+cmd+/"

### 放开 console 变量 注释
	"mac": "ctrl+alt+/"

### 删除 console 变量输出
  "key": "ctrl+shift+delete",
  "mac": "ctrl+cmd+delete"
